# Lab 5：個人隨手筆記本 (最終版)

這是本專案的 **最終整合版 (Lab 5)**！

在此階段，我們為 Telegram 小助手新增了 **個人隨手筆記本 (Note-Taking)** 功能。

小助手利用本地 SQLite 資料庫（`notes.db`）管理每位使用者的私有筆記。使用者可以新增筆記、條列查詢所有筆記，並能直接點選鍵盤上的按鈕來刪除個別項目。

---

## 🛠️ 新增指令說明

本階段新增了以下指令：
* `/addnote <內容>`：新增一筆隨手筆記。
  * 範例：`/addnote 下午記得買咖啡`
* `/notes`：查詢您儲存的所有筆記。
  * 會以清單條列顯示，並在底下自動生成對應編號的「🗑️ 刪除按鈕」（例如：`[🗑️ 1]`、`[🗑️ 2]`...）。
  * 點選按鈕後，機器人會直接從資料庫中刪除該筆記，並「無縫重新整理」對話訊息。

---

## 🚀 專案完整功能清單

到此階段，您的 Telegram 小助手已完全具備以下功能：
1. **基礎 Telegram Bot**：基礎 Echo 與連線監聽。
2. **依題目產生 PPT** (`/ppt`)：由 Gemini 設計大綱，`python-pptx` 動態排版 (支援 light/dark 配色)。
3. **依題目產生圖片** (`/draw`)：由 Imagen 3 生成高畫質圖片，支援 1:1, 16:9, 9:16 等多種比例。
4. **AI 圖片修改** (上傳照片)：上傳圖片至對話框，輸入口語指令讓 Gemini 與 Imagen 3 協同修改您的圖片。
5. **隨手筆記本** (`/addnote`, `/notes`)：本機 SQLite 隔離儲存，並支援 Inline 按鈕點選快速刪除。

## 🚀 本地執行步驟

### 1. 建立並啟用虛擬環境

進入 `lab5_note_taking` 資料夾：

```bash
# 建立虛擬環境
python3 -m venv venv

# 啟用虛擬環境 (macOS / Linux)
source venv/bin/activate

# 啟用虛擬環境 (Windows)
# .\venv\Scripts\Activate.ps1
```

### 2. 安裝依賴套件

```bash
pip install -r requirements.txt
```

### 3. 設定環境變數

將目錄下的 `.env.example` 複製為 `.env`，並填入您先前的各項 Token。

### 4. 啟動機器人

```bash
python bot.py
```

啟動後，Bot 會自動初始化本地 SQLite 資料庫 `notes.db` 用於儲存筆記。

## 🧪 測試隨手筆記

打開 Telegram，找到您的機器人：

1. **新增筆記**：
   發送 `/addnote 明天早上十點開會`。機器人會回應儲存成功。
   再發送 `/addnote 下午三點記得買冰美式`。
2. **列出筆記**：
   發送 `/notes`，機器人會回傳您的筆記清單，下方會有按鈕顯示為 `🗑️ 1` 和 `🗑️ 2`。
3. **刪除筆記**：
   點擊底下對應的刪除按鈕（例如點擊 `🗑️ 1`），您會發現訊息自動重新整理，原本編號為 1 的筆記被移除，且不需要產生新的對話訊息。
