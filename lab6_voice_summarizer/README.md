# Lab 6：語音轉譯與摘要筆記本 (最新整合版)

這是本專案的 **最新整合版 (Lab 6)**！

在此階段，我們為 Telegram 小助手新增了 **語音轉譯與摘要筆記本 (Voice Transcription & Summarization)** 功能。
小助手可以直接接收使用者的語音訊息（Voice Note），使用 Gemini 原生多模態音訊引擎進行轉譯與重點摘要，並支援寫入獨立的 `voice_notes` 本地 SQLite 資料表。同時，本 Lab 針對所有阻礙非同步事件循環的同步 I/O 與 CPU 密集操作（如資料庫存取、PPT 渲染、圖片處理等）進行了非同步（`asyncio.to_thread`）重構，並引入了嚴格的暫存檔案清理機制，防堵資源與記憶體洩漏。

---

## 🛠️ 新增指令與功能說明

### 🎙️ 語音辨識與摘要 (自動觸發)
* 直接向機器人傳送一個語音訊息（Voice Note）。
* 機器人會下載音訊，直接呼叫 Gemini 音訊理解模型，並回傳**逐字轉譯內容**與**核心摘要**。
* 訊息下方會附加一個 `💾 儲存至語音筆記` 的內聯按鈕。點選後將正式儲存至本機資料庫。
* 下載的音訊暫存檔會在處理完畢後被立即刪除，防範磁碟洩漏。

### 📂 語音筆記管理 (`/voicenotes`)
* 輸入 `/voicenotes`：查詢您儲存的所有語音筆記。
* 會以清單條列顯示，並在底下自動生成對應編號的控制按鈕：
  * `📄 編號`：點選後，機器人會以新訊息發送完整的語音轉譯原文。
  * `🔊 編號`：點選後，機器人會使用已記錄的 `file_id` 重播語音給您聽（不耗費伺服器儲存空間）。
  * `🗑️ 編號`：點選後，機器人會從資料庫中刪除該語音筆記，並無縫重新整理清單訊息。

## 🚀 專案完整功能清單

到此階段，您的 Telegram 小助手已完全具備以下功能：
1. **基礎 Telegram Bot**：基礎 Echo 與連線監聽。
2. **依題目產生 PPT** (`/ppt`)：由 Gemini 設計大綱，`python-pptx` 動態排版 (支援 light/dark 配色，並以 `asyncio.to_thread` 優化效能)。
3. **依題目產生圖片** (`/draw`)：由 Imagen 3 生成高畫質圖片，支援多種長寬比。
4. **AI 圖片修改** (上傳照片)：上傳圖片至對話框，輸入口語指令讓 Gemini 與 Imagen 3 協同修改您的圖片（具備重複上傳防護與暫存檔 100% 清理）。
5. **隨手文字筆記** (`/addnote`, `/notes`)：本機 SQLite 隔離儲存，支援按鈕快速刪除。
6. **語音轉譯筆記** (語音輸入, `/voicenotes`)：Gemini 原生音訊轉譯、重點摘要、語音重播、以及轉譯詳細檢視。

## 🚀 本地執行步驟

### 1. 建立並啟用虛擬環境

進入 `lab6_voice_summarizer` 資料夾：

```bash
# 建立虛擬環境 (使用 uv 可極速完成，若無安裝 uv 可用 python3 -m venv venv)
uv venv venv

# 啟用虛擬環境 (macOS / Linux)
source venv/bin/activate

# 啟用虛擬環境 (Windows)
# .\venv\Scripts\Activate.ps1
```

### 2. 安裝依賴套件

```bash
# 使用 uv 安裝 (若無 uv 請使用 pip install -r requirements.txt)
uv pip install -r requirements.txt
```

### 3. 設定環境變數

將目錄下的 `.env.example` 複製為 `.env`，並填入您的各項 Token。

### 4. 啟動機器人

```bash
python bot.py
```

啟動後，Bot 會自動初始化本地 SQLite 資料庫 `notes.db` 並自動生成 `notes` 與 `voice_notes` 兩張資料表。

## 🧪 測試語音轉譯與管理

打開 Telegram，找到您的機器人：

1. **傳送語音**：
   直接對機器人錄音並傳送。機器人會顯示下載及分析狀態，隨後回傳逐字稿與摘要，下方附有 `💾 儲存至語音筆記` 按鈕。
2. **儲存筆記**：
   點擊 `💾 儲存至語音筆記`。機器人會提示儲存成功，並移去該儲存按鈕。
3. **列出語音筆記**：
   發送 `/voicenotes`，機器人會回傳您的語音筆記清單，下方會有按鈕顯示為 `📄 1`、`🔊 1` 與 `🗑️ 1`。
4. **詳細原文、語音重播與刪除**：
   * 點擊 `📄 1`，會送出長文逐字稿。
   * 點擊 `🔊 1`，機器人會重新發送該語音給您。
   * 點擊 `🗑️ 1`，列表會原地刷新並刪除該筆記。
